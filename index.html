VoiceCal

What is VoiceCal?

VoiceCal is a web application that allows users to create Google Calendar events using voice commands.

How it works

Users sign in with their Google account. VoiceCal uses the Google Calendar API to create calendar events based on spoken input provided by the user.

What data is accessed?

VoiceCal requests permission to create calendar events in the user's Google Calendar. No data is stored on external servers.
