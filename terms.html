4. Data Usage

VoiceCal does not sell user data, does not share user data with third parties, and does not use Google user data for advertising, analytics, profiling, or any secondary purposes.

Google Calendar data is accessed solely to create, modify, or delete calendar events requested by the authenticated user. VoiceCal does not access data from any other Google services.

VoiceCal does not permanently store Google Calendar event data. Data is processed only as necessary to complete the requested action.
